## Для запуска необходимо

- Скачать папку [media](https://drive.google.com/file/d/17tzG67NKaxQrWtGYcQfhpsC1HgHIOitX/view?usp=sharing);
- Распаковать и положить в папку src;
- Выполнить команду
```
npm install
```
- Выполнить команду 
```
npm run serve
```
